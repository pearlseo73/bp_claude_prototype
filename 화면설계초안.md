# 자린고치 화면 설계서

## 1. 화면 설계 개요
자린고치는 가계부 관리, 예산 관리, 굴비 캐릭터 육성, 메이트 비교, 커뮤니티 기능을 포함하는 웹 서비스이다.  
본 문서는 핵심 화면의 구성 요소와 화면 간 이동 흐름을 정의한다.

---

## 2. 화면 목록

| 화면 ID | 화면명 | 설명 |
|---|---|---|
| P01 | 로그인 / 회원가입 화면 | 사용자 인증 |
| P02 | 메인 대시보드 | 오늘의 예산 현황, 굴비 상태, 주요 알림 요약 |
| P03 | 수입/지출 등록 화면 | 거래 내역 입력 |
| P04 | 거래 내역 목록 화면 | 수입/지출 내역 조회, 수정, 삭제 |
| P05 | 카테고리 관리 화면 | 카테고리 추가/수정/삭제 |
| P06 | 예산 설정 화면 | 주간/월간 예산 입력 및 자동 계산 결과 표시 |
| P07 | 예정 지출 관리 화면 | 예정 지출 등록, 자동 추정 결과 확인 |
| P08 | 통계 화면 | 수입/지출 통계 그래프 조회 |
| P09 | 주간 레포트 화면 | 주간 소비 요약 및 피드백 |
| P10 | 월간 레포트 화면 | 월간 소비 요약, 다짐 작성, 굴비 응답 |
| P11 | 굴비 캐릭터 화면 | 굴비 상태, 체중, 성격, 성장 상태 확인 |
| P12 | 옷장 화면 | 보유 의상 목록 및 착용 |
| P13 | 새끼 굴비 화면 | 첫째/둘째 새끼 상태 확인 |
| P14 | 메이트 화면 | 메이트 요청/수락/해제, 옆집 굴비 비교 |
| P15 | 커뮤니티 목록 화면 | 게시글 목록 조회 |
| P16 | 커뮤니티 작성/상세 화면 | 게시글 작성, 댓글, 좋아요 |
| P17 | 대화 로그 화면 | 월간 레포트에서 생성된 굴비와의 대화 조회 |

---

## 3. 화면 상세 설계

### P01. 로그인 / 회원가입
#### 목적
- 사용자가 서비스에 로그인하거나 회원가입한다.

#### 주요 요소
- 아이디 입력
- 비밀번호 입력
- 로그인 버튼
- 회원가입 버튼

#### 이동
- 로그인 성공 → P02 메인 대시보드
- 회원가입 완료 → P02 메인 대시보드 또는 로그인 화면

---

### P02. 메인 대시보드
#### 목적
- 사용자가 전체 상태를 한눈에 확인한다.

#### 주요 요소
- 이번 주 예산
- 현재까지 지출 금액
- 예산 대비 지출 퍼센트
- 운용 가능 금액
- 오늘의 굴비 상태
- 굴비 알림 메시지
- 빠른 이동 버튼
  - 거래 등록
  - 거래 내역
  - 예산 설정
  - 레포트 보기
  - 굴비 보기

#### 이동
- 거래 등록 버튼 → P03
- 거래 내역 버튼 → P04
- 예산 설정 버튼 → P06
- 굴비 보기 버튼 → P11

---

### P03. 수입/지출 등록 화면
#### 목적
- 사용자가 수입 또는 지출을 등록한다.

#### 주요 요소
- 거래 유형 선택(수입/지출)
- 금액 입력
- 카테고리 선택
- 날짜 선택
- 메모 입력
- 저장 버튼

#### 동작
- 지출 저장 시 주간 예산 대비 퍼센트 계산
- 굴비 알림 문구 생성

#### 이동
- 저장 완료 → P04 거래 내역 목록 또는 P02 대시보드

---

### P04. 거래 내역 목록 화면
#### 목적
- 사용자가 기존 거래 내역을 조회, 수정, 삭제한다.

#### 주요 요소
- 기간 필터
- 카테고리 필터
- 거래 목록 테이블
- 수정 버튼
- 삭제 버튼

---

### P05. 카테고리 관리 화면
#### 목적
- 사용자가 카테고리를 관리한다.

#### 주요 요소
- 카테고리 목록
- 카테고리 추가
- 카테고리 수정
- 카테고리 삭제

---

### P06. 예산 설정 화면
#### 목적
- 사용자가 주간 또는 월간 예산을 설정한다.

#### 주요 요소
- 입력 방식 선택(주간/월간)
- 예산 금액 입력
- 자동 계산 결과 표시
  - 주간 예산
  - 월간 예산
- 저장 버튼

#### 정책
- 주간 예산 입력 시 월간 예산 자동 계산
- 월간 예산 입력 시 주간 예산 자동 계산
- 서비스의 핵심 관리 기준은 주간 예산

---

### P07. 예정 지출 관리 화면
#### 목적
- 사용자가 예정 지출을 등록하거나 자동 추정 내역을 확인한다.

#### 주요 요소
- 예정 지출 등록 폼
- 자동 추정 예정 지출 목록
- 확정 버튼
- 삭제 버튼
- 운용 가능 금액 표시

---

### P08. 통계 화면
#### 목적
- 사용자가 수입/지출 통계를 시각적으로 확인한다.

#### 주요 요소
- 월별 수입 그래프
- 월별 지출 그래프
- 카테고리별 수입 비율 그래프
- 카테고리별 지출 비율 그래프
- 예산 대비 지출 현황 그래프

---

### P09. 주간 레포트 화면
#### 목적
- 사용자가 이번 주 소비를 확인하고 피드백을 받는다.

#### 주요 요소
- 이번 주 소비 합계
- 저번 주와 비교
- 굴비 피드백 메시지
- 예산 성공 여부

---

### P10. 월간 레포트 화면
#### 목적
- 사용자가 월간 소비를 회고하고 굴비와 상호작용한다.

#### 주요 요소
- 이번 달 소비 요약
- 이전 달과 비교
- 굴비 성장 반영 결과
- 메이트와 비교 요약
- 다짐 입력 칸
- 굴비 응답 영역
- 저장 버튼

#### 정책
- 굴비와의 대화는 월간 레포트에서만 가능

---

### P11. 굴비 캐릭터 화면
#### 목적
- 사용자가 내 굴비의 상태를 확인한다.

#### 주요 요소
- 선택한 굴비 종류
- 성격 요약
- 현재 외형
- 체중 수치
- 오늘의 상태
- 성장 조건 진행 현황

---

### P12. 옷장 화면
#### 목적
- 사용자가 획득한 의상을 확인하고 착용한다.

#### 주요 요소
- 보유 의상 목록
- 잠금 의상 목록
- 착용 버튼
- 현재 착용 의상 표시

---

### P13. 새끼 굴비 화면
#### 목적
- 사용자가 새끼 굴비의 상태를 확인한다.

#### 주요 요소
- 첫째 새끼 존재 여부
- 첫째 새끼 체중
- 둘째 새끼 존재 여부
- 둘째 새끼 체중
- 알 게이지 또는 성장 표시

---

### P14. 메이트 화면
#### 목적
- 사용자가 메이트를 관리하고 비교한다.

#### 주요 요소
- 메이트 요청
- 받은 요청 목록
- 수락 / 거절 버튼
- 메이트 해제 버튼
- 옆집 굴비 상태 비교
  - 성인 굴비 체중
  - 새끼 존재 여부
  - 새끼 체중
  - 의상 보유 현황

---

### P15. 커뮤니티 목록 화면
#### 목적
- 커뮤니티 게시글을 조회한다.

#### 주요 요소
- 게시글 목록
- 좋아요 수
- 베스트 고치 영역
- 글쓰기 버튼

---

### P16. 커뮤니티 작성/상세 화면
#### 목적
- 사용자가 게시글을 작성하거나 게시글 상세를 본다.

#### 주요 요소
- 제목
- 내용
- 고치 이미지/상태 표시
- 댓글 목록
- 댓글 작성
- 좋아요 버튼

---

### P17. 대화 로그 화면
#### 목적
- 사용자가 월간 레포트에서 굴비와 주고받은 대화를 다시 확인한다.

#### 주요 요소
- 월별 대화 목록
- 사용자 다짐
- 굴비 응답

---

## 4. 주요 사용자 흐름

### 흐름 1. 거래 등록
P02 대시보드 → P03 거래 등록 → 저장 → 예산 퍼센트 계산 및 굴비 알림 → P04 또는 P02

### 흐름 2. 예산 설정
P02 → P06 예산 설정 → 주간/월간 입력 → 자동 계산 → 저장

### 흐름 3. 월간 회고
P02 → P10 월간 레포트 → 다짐 작성 → 굴비 응답 확인 → P17 대화 로그 저장

### 흐름 4. 굴비 성장 확인
P02 → P11 굴비 상태 → P12 옷장 → P13 새끼 상태

### 흐름 5. 메이트 비교
P02 → P14 메이트 화면 → 요청/수락 → 옆집 굴비 확인

### 흐름 6. 커뮤니티 활동
P02 → P15 게시글 목록 → P16 게시글 상세 → 댓글/좋아요
